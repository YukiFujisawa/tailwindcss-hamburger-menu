# Tailwind CSS で作成したハンバーガーメニュー(レスポンシブ対応済み)

## 特徴

1. jQuery を使わず TailwindCSS で作成したハンバーガーメニューです。
2. レスポンシブ対応(スマホ画面での最適化)済みです。
   - 640px まで：ハンバーガーメニュー
   - 640px 以上：通常のメニュー
